## Функциональное тестирование (Cypress)

Проект использует Cypress для функционального тестирования пользовательских сценариев.

### Запуск Cypress тестов

#### Интерактивный режим (с GUI)

```bash
npm run cypress:open
```

Эта команда откроет окно Cypress Test Runner, где вы сможете:
- Выбрать браузер для тестирования
- Запускать тесты в интерактивном режиме
- Видеть выполнение тестов в реальном времени
- Отлаживать тесты

#### Запуск в headless режиме (CI/CD)

```bash
npm run cypress:run
```

### Запуск тестов

```bash
# Запуск конкретного файла тестов
npx cypress run --spec "cypress/e2e/constructor.cy.ts"

# Запуск в конкретном браузере
npx cypress run --browser chrome
```

### Важные замечания-уточнения

- Перед запуском тестов убедитесь, что приложение запущено на `http://localhost:3000`
- Тесты используют моки API, поэтому не требуют реального сервера
- Для тестирования drag and drop используется симуляция событий браузера
